Синтезатор речи

Автор: Султангулов Денис Ринатович

    Пользователь может прослушать какой-либо текст по буквам, которые записаны в базе данных.
    При включенной кнопке  “Синтез речи в реальном времени”, пользователь, вводя текст с клавиатуры, услышит введённую букву
    Присутствует возможность изменить скорость воспроизведения в большую и меньшую сторону.
    Нажав кнопку “Воспроизвести”, все введённые буквы воспроизведется.
    Если пользователь хочет прослушать слово по слогам ему необходимо перейти на вкладку запись слога/буквы, ввести нужный слог слова, затем нажать кнопку записи, произнести слог и завершить запись. Пользователь может посмотреть слоги слова во вкладке с записью. После этого записанный слог попадёт в базу данных.
    Присутствует возможность прослушать записанный слог.
    С помощью выпадающего меню пользователь может сменить синтез речи на гугловского робота.
    На вкладке “База данных” пользователь может управлять базой данных. За отображение всех записей отвечает кнопка “Вывести всё”. Чтобы вывести или удалить нужный слог/букву, следует написать слог/букву и нажать кнопку “Вывести” или “Удалить”.
